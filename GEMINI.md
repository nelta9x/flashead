# Gemini CLI Instructions

이 프로젝트에서는 다음 문서들에 정의된 규칙과 지침을 최우선으로 준수합니다.

- **[CLAUDE.md](./CLAUDE.md)**: 프로젝트 개요, 기술 스택, 주요 명령어, 코드 규칙 및 데이터 시스템 구조가 정의되어 있습니다.
- **[CODEMAP.md](./CODEMAP.md)**: 프로젝트 전체 구조, 시스템 간의 상호작용 및 이벤트 흐름(EventBus) 정의. 새로운 기능 구현 전 반드시 참조하십시오.
- **[LESSONS.md](./LESSONS.md)**: 과거의 버그 사례와 교훈이 기록되어 있습니다. 새로운 코드를 작성하거나 수정할 때 유사한 실수를 반복하지 않도록 반드시 참조하십시오.

## 핵심 준수 사항
1. 작업을 시작하기 전에 위 문서들을 먼저 읽고 맥락을 파악하십시오. 특히 시스템 간 연동이나 대규모 수정 시 `CODEMAP.md`를 나침반으로 삼으십시오.
2. `CLAUDE.md`에 명시된 코드 스타일과 `console.log` 제거 규칙을 엄격히 따르십시오.
3. 모든 게임 밸런스 및 설정 수정은 `data/`의 JSON 파일을 통해 수행하십시오.
4. **문서 동기화**: 새로운 시스템 추가, 렌더러 분리, 이벤트 추가 등 구조적 변화가 있을 경우 반드시 `CODEMAP.md`를 최신화하여 나침반의 정확성을 유지하십시오.
5. **절대 원칙: 모든 테스트가 통과(`npm run test:run`)하기 전에는 어떠한 변경 사항도 커밋하지 마십시오.** 테스트 실패 시 반드시 문제를 해결한 후 다시 검증하십시오.
